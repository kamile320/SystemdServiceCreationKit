# SystemdServiceCreationKit
A written in Python tool to easily create systemd services.

Easily create systemd services using SSCK - in this script you'll be asked for data like: 
- Working (main) directory
- Name of file
- In what programming language is your script written
- (If python) Do you use virtual environment or not
- Should your service be restarted on fail
- etc.

This project will be based on [systemd_add.py](https://github.com/kamile320/ServerBot/blob/main/Files/systemd_add.py).
